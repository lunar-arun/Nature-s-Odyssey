## 📌 Project Overview

**Nature's Odyssey** is a carbon footprint awareness platform designed to transform climate awareness into meaningful action. Developed for **Challenge 3**, the application helps users calculate their environmental impact, discover sustainable lifestyle improvements, and track daily habits that contribute to reducing carbon emissions over time.

## 🚀 Core Features

- **Personalized Carbon Calculator:** Analyze transportation, energy consumption, diet, and lifestyle habits to estimate your annual carbon footprint.
- **Interactive Dashboard:** Visualize carbon footprint data and compare it with average benchmarks for better understanding.
- **Habit Tracker:** Monitor eco-friendly activities and measure their impact on reducing emissions.
- **Smart Recommendations:** Receive personalized suggestions focused on the areas contributing most to your carbon footprint.
- **Progress Monitoring:** Track improvements over time and stay motivated with measurable results.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Data Visualization:** Chart.js
- **Backend:** Node.js
- **Deployment:** Render

## 📦 Installation & Local Setup

### Prerequisites

- Install [Node.js](https://nodejs.org/)

### 1. Clone the Repository

```bash
git clone https://github.com/lunar-arun/Nature-s-Odyssey.git
cd Nature-s-Odyssey
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables (Optional)

Create a `.env` file in the project root if required:

```env
PORT=5000
DATABASE_URL=your_database_connection_string
```

### 4. Run the Application

```bash
npm start
```

Open your browser and visit:

```text
http://localhost:3000
```

## 📊 Carbon Footprint Calculation

The platform estimates emissions using standard carbon conversion factors:

- **Transportation:** `Distance Traveled × Emission Factor = CO₂ Emissions`
- **Energy Usage:** `Electricity Consumption × Grid Emission Factor = CO₂ Emissions`
- **Dietary Impact:** Emissions are adjusted based on food consumption patterns.
- **Lifestyle Activities:** Daily habits contribute to personalized carbon reduction estimates.
